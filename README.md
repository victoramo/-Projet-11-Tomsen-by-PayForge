📝 Description
Élaboration d'une stratégie de test complète dans un contexte Agile pour le projet Tomsen by PayForge. Ce projet combine gestion de projet et assurance qualité.

Structurer une démarche qualité globale adaptée aux enjeux métier, en intégrant les pratiques agiles (sprints, revue des exigences).

🛠️ Travail réalisé
Revue des exigences et planification des sprints

Rédaction du cahier de recette et des scénarios de test
